Dress Drizzy
Click here to visit the site

Dress Drizzy is a fun web app I created in the middle of the night sometime during my sophomore spring semester. I was supposed to teach JavaScript basics to my web design decal class and I thought I'd use this to demonstrate some of the cool things you can build with JavaScript.

For this project, I used Adobe Illustrator's image trace function to create the cartoonish look. Adobe Photoshop and Zara's men's catalogue were used to create the different outfits.
